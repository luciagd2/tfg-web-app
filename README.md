# TFG - Desarrollo de una aplicación web para la gestión y distribución
de patrones de crochet y ganchillo utilizando los Framewoks
Bootstrap y Spring Boot

Este proyecto es un Trabajo de Fin de Grado (TFG) para la Universidad de Alcalá (Alcalá de Henares, Madrid, España) realizado por [Lucía Patricia Gil Díaz](https://www.linkedin.com/in/lucía-patricia-gil-diaz-065b85266/) y dirigido por Roberto Barchito Plata.

El objetivo principal es desarrollar una aplicación web dedicada al crochet y el ganchillo, enfocada en compartir y gestionar patrones de forma colaborativa.

## Tecnologías utilizadas

- **Frontend:** HTML, CSS, JavaScript, Bootstrap  
- **Backend:** Java, Spring Boot  
- **Base de datos:** MySQL (Azure MySQL Database)  
- **Despliegue:** Azure App Service  
- **Herramientas:** GitHub, MySQL Workbench
